# node-wiring-bpi-zero
BPI-M2 ZERO 용 node.js gpio 패키지

# 설치
$ cd ~<br/>
$ git clone https://github.com/bluexmas/node-wiring-bpi-zero.git<br/>
$ npm install node-wiring-bpi-zero

# 데모 실행
$ sudo node node-wiring-bpi-zero/examples/blink.js
